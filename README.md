# Forwarder-Mod of nx-hbloader
Host process for loading Switch homebrew NROs. this mod allows to create NROs/Reroarch Forwarders. 

# Notes
These are patches not made by me, but rather collected from previous patches made to create forwarders and re-applied to the latest hbl loader.

https://github.com/switchbrew/nx-hbloader
https://github.com/Skywalker25/Forwarder-Mod
